🚀 Guia Básico de Comandos Git

Configuração Inicial

Configurar nome do usuário git config --global user.name "Seu Nome"
Configurar e-mail git config --global user.email "seu@email.com"
Ver configurações git config --list

Criar e Clonar Repositórios

Iniciar repositório Git git init
Clonar repositório git clone URL_DO_REPOSITORIO

Trabalhando com Arquivos

Ver status dos arquivos git status
Adicionar arquivo específico git add arquivo.txt
Adicionar todos os arquivos git add .
Criar commit git commit -m "Mensagem do commit"
Ver histórico de commits git log
Histórico resumido git log --oneline

Branches

Listar branches git branch
Criar branch git branch nova-branch
Trocar de branch git checkout nome-branch
Criar e trocar de branch git checkout -b nova-branch
Mesclar branch git merge nome-branch

GitHub

Conectar repositório remoto git remote add origin URL
Enviar para o GitHub git push -u origin main
Enviar alterações git push
Baixar alterações git pull
Buscar alterações sem baixar git fetch

Desfazer Alterações

Remover arquivo da área de stage git reset arquivo.txt
Voltar ao último commit git reset --hard HEAD
Reverter commit git revert ID_DO_COMMIT

Informações Extras

Ver repositórios remotos git remote -v
Ver diferenças git diff
Ver ajuda git help
